# Workflow

## Configure
Create `lserve` configuration in current directory.

```shell
lserve init
```
1. Create .lserve directory
2. Setup workflows `.lserve/workflow.json` file and `.lserve/workflow` directory
3. Install workflow plugins and middlewares.


## Setup

```shell
lserve setup
```
